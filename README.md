## 📋 About the Project

FSV Capital Startup Funding Platform is a full-stack, investor-grade 
digital application system for startups seeking funding. Built as part 
of the AI Developer Intern assignment for Future Transformation Pvt Ltd.

The platform enables:
- Structured deal intake from startup founders
- Investor-grade screening with smart filtering
- Automated deal scoring (0–100)
- Pipeline creation for DeepTech, Fintech, AI, Blockchain startups
